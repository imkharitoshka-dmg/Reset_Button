# Reset Button — Development Guide

## Что это

Reset Button — Android-приложение на Flutter для коротких reset-сценариев: усталость, тревога, перегруз, потеря фокуса, хаос в голове, подготовка к встрече, восстановление после тяжёлого разговора.

## Технические ограничения MVP

- Android-приложение.
- Flutter.
- Русский интерфейс.
- Без бэкенда.
- Без авторизации.
- Без Firebase.
- Без интернета.
- Без уведомлений.
- Без таймера в MVP.
- Локальное хранение через shared_preferences.
- Данные сценариев зашиты локально в приложении.
