# Reset Button — Development Guide

## Что это

Reset Button — Android-приложение на Flutter для коротких reset-сценариев на 3, 5 и 10 минут. Приложение помогает сделать короткую паузу, пройти локальный чек-лист, отметить результат и сохранить историю на устройстве.

Текущий flow:

```text
главный экран -> кластер -> состояние -> длительность -> чек-лист -> результат -> заметка -> история
```

На главном экране используются 5 кластеров: `Успокоиться`, `Восстановиться`, `Разгрузить голову`, `Сфокусироваться`, `Переключиться после общения`.

## Технические ограничения MVP

- Android-приложение.
- Flutter.
- Русский интерфейс.
- Без бэкенда.
- Без авторизации.
- Без Firebase.
- Без интернета.
- Без уведомлений.
- Без таймера в MVP.
- Без AI API.
- Без RAG.
- Локальное хранение через shared_preferences.
- Данные сценариев зашиты локально в приложении.
- Controlled rotation работает локально на истории reset-сессий.
