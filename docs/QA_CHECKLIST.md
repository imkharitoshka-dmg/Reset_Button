# Reset Button — QA Checklist перед APK

## A. Базовые проверки

```bash
flutter analyze
flutter test
flutter run -d emulator-5554
```

- Приложение открывается.
- Главный экран отображает 5 кластеров.
- В каждом кластере 3 состояния.
- Выбор длительности 3 / 5 / 10 минут работает.
- Чек-лист открывается.
- Результат сохраняется.
- Заметка сохраняется.
- История сохраняется после перезапуска.

## B. Controlled rotation / routing

- Для одной комбинации состояние + длительность пройти сценарий несколько раз.
- Убедиться, что при наличии альтернатив следующий запуск не повторяет последний `scenarioVariantId`.
- Проверить, что cold start использует default-сценарий.
- Проверить, что `scenarioVariantId` сохраняется в историю.
- Проверить, что старые записи без `scenarioVariantId` не ломают историю.

## C. Ranking

- Сценарий с результатом `помогло` получает больший вес.
- `частично` учитывается слабее, чем `помогло`.
- `не помогло` не усиливает сценарий.
- Новые и редко использованные сценарии получают `explorationBonus`.
- Default-сценарий получает небольшой `defaultPriorityBonus`.
- При равных `score` выбор стабильный.

## D. История и аналитика

- История показывает сценарий/вариант, если `scenarioVariantId` найден.
- Технический `scenarioVariantId` не показывается пользователю.
- Блок `Что чаще помогало` скрыт до 5 завершённых сценариев.
- После 5 завершённых сценариев блок появляется.
- По клику открывается статистика.
- Топ состояний считается по частоте использования.
- Детализация по состоянию показывает сценарии и счётчики `помогло / частично / не помогло`.
- Старые названия состояний нормализуются только для аналитики.

## E. Ограничения MVP

- Нет Firebase.
- Нет бэкенда.
- Нет авторизации.
- Нет интернета.
- Нет уведомлений.
- Нет таймера.
- Нет AI API.
- Нет RAG.
- Нет новых зависимостей без явного решения.

## F. Release APK

```bash
flutter build apk --release
```

- APK не коммитить в репозиторий как обычный файл.
- APK можно передать на Android-телефон через Google Drive / Telegram / другой файловый канал.
- После установки на телефон проверить базовый flow и историю.
